# BJUTThesis

## 欢迎使用上北京工业大学论文模板

模板为撰写北京工业大学博士学位论文的非官方LaTeX 模板。

本模板目前支持 XeTeX 引擎，字符编码仅支持 UTF-8。



* 感谢 [TeX](http://www.ctan.org) 的开发人员和贡献者，以及 [CTeX](http://www.ctex.org/HomePage) 提供的 LaTeX 中文支持。
* 感谢 [部分校友](http://yzlab.net/BjutThsis.html) 早期的工作为模板制作提供了部分参考。
